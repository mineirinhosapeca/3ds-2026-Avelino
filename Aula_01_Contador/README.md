# 🚀 3DS-2026-Avelino: Desenvolvimento de Sistemas

Bem-vindos ao repositório central da disciplina de Desenvolvimento de Sistemas para o **3º Ano do Ensino Médio Técnico**. Este espaço foi criado para centralizar nossos exemplos, exercícios e materiais de aula.

## 🎯 Objetivo
Este repositório serve como um guia prático para consolidar conceitos de lógica de programação, versionamento de código e boas práticas de desenvolvimento que discutimos em sala.

## 📂 O que você encontra aqui?
* **Lógica com Python:** Exemplos práticos (como o jogo de adivinhação) explorando loops `while True`, tratamento de erros com `try/except` e condicionais.
* **Estruturas Front-end:** Exemplos em HTML5 e CSS3 para criação de interfaces modernas.
* **Versionamento:** Demonstração real de uso do Git e fluxos de trabalho no GitHub.

## 🛠️ Tecnologias Utilizadas
* **Linguagens:** Python 3.x, HTML5, CSS3.
* **Ferramentas:** VS Code, Git, GitHub.

## 📋 Como utilizar este material?
1.  **Clone o repositório:** `git clone https://github.com/jerog1971/3ds-2026-Avelino.git`
2.  **Explore as pastas:** Cada aula ou exemplo terá seu próprio diretório organizado.
3.  **Dúvidas?** Abra uma *Issue* ou entre em contato durante as aulas!
